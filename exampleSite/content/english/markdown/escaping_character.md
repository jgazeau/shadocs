# Escaping characters
---

To escape characters, **prefix them with a backslash (\\)** (The list of characters that can be escaped is available below).

| Markdown | HTML | Rendering |
| -------- | ---- | --------- |
|{{< md >}}
```
\\
```
{{< /md >}}|{{< plaintext >}}
<p>\<p>
{{< /plaintext >}}|{{< md >}}
\\
{{< /md >}}|
|{{< md >}}
```

\*
```

|{{< md >}}
