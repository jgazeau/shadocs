# Comment
---

To define a comment, **use the same syntax as HTML comments (\<\!\-\- [...] \-\-\>)**.

| Markdown | HTML | Rendering |
| -------- | ---- | --------- |
|{{< md >}}
```
One see me
<!-- One doesn't see me -->
```
{{< /md >}}|{{< plaintext >}}
<p>One see me<p>
<!-- One doesn't see me -->
{{< /plaintext >}}|{{< md >}}
One see me
<!-- One doesn't see me -->
{{< /md >}}|
