# Paragraphe
---

Pour définir un paragraphe, il faut **séparer les lignes de textes par 1 ligne vide**.

| Markdown | HTML | Rendu |
| -------- | ---- | ----- |
|{{< md >}}
```
Je suis un paragraphe.

Et moi un autre !
```
{{< /md >}}|{{< plaintext >}}
<p>Je suis un paragraphe.</p>
<p>Et moi un autre !</p>
{{< /plaintext >}}|{{< md >}}
Je suis un paragraphe.

Et moi un autre !
{{< /md >}}|
