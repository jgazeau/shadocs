# Description
---

Par défaut le thème gère le [mode multilangue de Hugo](https://gohugo.io/content-management/multilingual/#readout).

{{< alert type="warning" >}}
Un menu situé dans la barre de navigation permet de basculer vers la langue choisie, suivant la configuration du site. Toutefois ce menu n'apparaîtra que si la page courante dispose d'une traduction (***la page doit être nommée de la même manière dans toutes les traductions***).
{{< /alert >}}
